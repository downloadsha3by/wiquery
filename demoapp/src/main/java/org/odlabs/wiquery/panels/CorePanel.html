WiQuery behaviors

WiQuery aims to ease writing of JavaScript with Wicket, e.g. use behaviors to attach jQuery statements to a component.

Example 1: change border style when a click happens

This first example describes how to generate a simple jQuery statement.

Example 1 !

<span wicket:id="example1">Example 1 !</span>

// consider being in a WebPage constructor public MyWebPage(String id) { super(id); Label example1 = new Label("example1", "Example 1"); example1.add(new WiQueryEventBehavior(new Event(MouseEvent.CLICK) { @Override public JsScope callback() { return new JsScope() { @Override protected void execute(JsScopeContext scopeContext) { scopeContext.self().chain(CssHelper.css("border", "1px solid red")); } }; } })); this.add(example1); }
