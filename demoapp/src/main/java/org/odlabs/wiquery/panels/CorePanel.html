Core jQuery statements

WiQuery aims to be closed to jQuery programming style. That's why WiQuery implements the jQuery API with the same "fluent interface" principle.

Example 1: change border style when a click happends

This first example describes how to generate a simple jQuery statement.

// consider being in a WebPage constructor public MyWebPage() { Label example1 = new Label("example1", "Example 1"); example1.add(new WiQueryEventBehavior(new Event(MouseEvent.CLICK) { @Override public JsScope callback() { return new JsScope() { @Override protected void execute(JsScopeContext scopeContext) { scopeContext.self().chain(CssHelper.css("border", "1px solid red")); } }; } })); add(example1); }
