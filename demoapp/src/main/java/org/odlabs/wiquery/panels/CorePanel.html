WiQuery behaviors

WiQuery aims to easy writting JavaScript with Wicket, e.g. use behaviors to attache jQuery statements to a component.

Example 1: change border style when a click happends

This first example describes how to generate a simple jQuery statement.

Example 1 !

<span wicket:id="example1">Example 1 !</span>

public MyWebPage() { Label example1 = new Label("example1", "Example 1"); example1.add(new WiQueryEventBehavior(new Event(MouseEvent.CLICK) { @Override public JsScope callback() { return new JsScope() { @Override protected void execute(JsScopeContext scopeContext) { scopeContext.self().chain(CssHelper.css("border", "1px solid red")); } }; } })); add(example1); }
