Core jQuery statements

WiQuery aims to be closed to jQuery programming style. That's why WiQuery implements the jQuery API with the same "fluent interface" principle.

WiQuery comes with severals helpers to bind the whole jQuery API, here are some examples:

Example: CSS Helper

This example describes how to dynamically change CSS with WiQuery. WiQuery ease jQuery integration and provides behaviors to do it with Wicket style.

This first example shows how to attach a simple jQuery CSS statement, e.g. dynamically style an element with JavaScript.

Example 1 !

<span wicket:id="example1">Example 1 !</span>

// consider being in a WebPage constructor public MyWebPage() { final Label label = new Label("example1", "CSS Helper curtomized me"); chilliPlugin.add(label); label.add(new WiQueryAbstractBehavior() { @Override public JsStatement statement() { return new JsQuery(label).$().chain(CssHelper.css("border", "1px solid red")); } }); }

This example introduces the WiQueryAbstractBehavior class, which is used to attached a jQuery statement to a component. The JsStatement class defines a jQuery statement (e.g. what will be generated by our engine).

The JsQuery class is there to initialize a JsStatement properly and to attach it to a Wicket component. Let's take a closer look at the querying code: public JsStatement statement() { return new JsQuery(label).$().chain(CssHelper.css("border", "1px solid red")); }

The new JsQuery(label) aims to create JsStatements for a given component. All of these statements can be chained with other statements to provide a fluent way to write JavaScript code from Wicket.

A JsQuery provides the well know jQuery $ method. Consider that a call to the $() method will generate the corresponding jQuery statement. If a component is attached to a JsQuery (like the one we wrote before), this will generate $("#thegeneratedcomponentid").

$().chain(CssHelper.css("border", "1px solid red")) will chain the corresponding jQuery statement to style elements.

WiQuery provides some helpers to bind core jQuery statements.
