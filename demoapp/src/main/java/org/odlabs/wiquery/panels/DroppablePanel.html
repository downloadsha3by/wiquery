WiQuery UI::Droppable

This page shows how droppable elements are bound with WiQuery.

Example 1: Set an element droppable

This example shows how to turn an element droppable, and how to declare where elements can be dropped.

Drop elements here

<div class="draggable" wicket:id="example1">Drag me !</div> <div class="dropzone" wicket:id="dropzone">Drop elements here</div>

public MyWebPage(String id) { super(id); Label label = new Label("example1", "Drag me !"); label.add(new DraggableBehavior()); this.add(label); WebMarkupContainer dropzone = new WebMarkupContainer("dropzone"); DroppableBehavior droppableBehavior = new DroppableBehavior(); droppableBehavior.setOnDrop(JsScope.quickScope("alert('dropped');")); droppableBehavior.setAccept(".draggable"); dropzone.add(droppableBehavior); this.add(dropzone); }
