WiQuery UI::Draggable

This page shows how draggable are bound with WiQuery.

Example: Set an element draggable

This first example shows how an element can be turned draggable.

Drag me !

<div wicket:id="example1">Drag me !</div>

public MyWebPage() { Label label = new Label("example1", "Drag me !"); add(label); label.add(new DraggableBehavior()); }
