WiQuery UI::Draggable

This page shows how draggable elements are bound with WiQuery.

Example: Set an element draggable

This first example shows how an element can be turned draggable.

Drag me !

<div wicket:id="example1">Drag me !</div>

// consider being in a WebPage constructor public MyWebPage(String id) { setup(id); Label label = new Label("example1", "Drag me !"); this.add(label); label.add(new DraggableBehavior()); }
